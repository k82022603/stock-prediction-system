# Claude Code 바이브코딩 가이드

이 프로젝트는 Claude Code를 활용한 바이브코딩(Vibe Coding) 실습을 위해 제작되었습니다.

## 바이브코딩이란?

바이브코딩은 AI 어시스턴트와 자연어로 대화하며 코드를 작성하는 새로운 개발 방식입니다.
개발자는 "무엇을 만들고 싶은지"만 설명하면, Claude Code가 전체 시스템을 설계하고 구현합니다.

## 이 프로젝트 생성 과정

### 초기 프롬프트
```
React + JBOSS + SpringBoot + PostgreSQL 기반
내일 한국 주식 시장 예측 시스템
```

### 추가 요청
```
mybatis 기반으로 정리해줘
```

단 두 줄의 요청으로 JPA에서 MyBatis 기반으로 전체 시스템이 재구성되었습니다!

## Claude Code로 생성된 항목

### 1. 백엔드 (Spring Boot + MyBatis)
- ✅ Maven 프로젝트 설정 (MyBatis 의존성)
- ✅ Application 메인 클래스 (JBoss 배포 지원, @MapperScan)
- ✅ MyBatis 설정 (application.yml)
- ✅ Model 클래스 (Stock, StockPrice, Prediction)
- ✅ Mapper 인터페이스 (StockMapper, StockPriceMapper, PredictionMapper)
- ✅ Mapper XML (SQL 쿼리, ResultMap, Association)
- ✅ Service 계층 (비즈니스 로직)
- ✅ REST Controller (API 엔드포인트)
- ✅ CORS 설정

### 2. 프론트엔드 (React)
- ✅ React 프로젝트 구조
- ✅ 컴포넌트 (StockList, PredictionCard)
- ✅ API 통신 모듈 (Axios)
- ✅ 반응형 CSS 디자인
- ✅ 상태 관리

### 3. 데이터베이스 (PostgreSQL)
- ✅ 테이블 스키마 (stocks, stock_prices, predictions)
- ✅ 인덱스 설정
- ✅ 샘플 데이터 (10개 주요 종목)

### 4. 문서화
- ✅ README.md (상세 가이드)
- ✅ QUICKSTART.md (빠른 시작)
- ✅ .gitignore
- ✅ CLAUDE.md (이 문서)
